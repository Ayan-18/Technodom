# Technodom Platform

**Technodom Platform** — это пет-проект для создания и управления онлайн-магазином. Платформа предоставляет функционал для просмотра товаров, оформления заказов, управления корзиной, а также включает административные возможности, такие как управление товарами, пользователями и заказами. Дополнительно реализованы функции отзывов, рассылки уведомлений об акциях и асинхронной обработки задач с использованием брокера сообщений.

Цель проекта — изучение технологий веб-разработки с использованием Go, базового HTML и брокера сообщений для реализации асинхронных операций.

---

## Цели проекта 

- Изучение базовых принципов разработки онлайн-магазинов.
- Реализация бизнес-логики, включая управление товарами, заказами, корзиной и пользователями.
- Асинхронная обработка задач, таких как рассылка уведомлений и обработка заказов.
- Настройка работы с брокером сообщений для повышения производительности и масштабируемости.
- Освоение работы с базами данных для хранения информации о товарах, пользователях, заказах и отзывах.
- Настройка контейнеризации с помощью Docker.

---

## Основные возможности

### Для пользователей:
- **Просмотр товаров**: каталог с фильтрацией, сортировкой и поиском.
- **Корзина**: добавление, удаление товаров, изменение их количества.
- **Оформление заказа**: создание и подтверждение заказов с уведомлением о статусе.
- **Оставление отзывов**: возможность оставлять отзывы и оценки для товаров.
- **Авторизация**: регистрация, вход и управление профилем.

### Для администраторов:
- **Управление товарами**: добавление, редактирование и удаление позиций.
- **Управление пользователями**: блокировка пользователей, управление их ролями.
- **Управление заказами**: просмотр, обработка и изменение статусов заказов.
- **Рассылка уведомлений**: автоматическая отправка сообщений через брокер сообщений (например, оповещения о скидках, статусе заказа и т.д.).

### Дополнительные функции:
- **Отзывы и рейтинги**: улучшение взаимодействия пользователей с продуктами.
- **Рассылка акций**: автоматическая и ручная рассылка уведомлений о скидках и специальных предложениях.
- **Система уведомлений**: оповещения о статусе заказов и новостях через брокер сообщений.
- **Асинхронная обработка задач**: обеспечение высокой производительности при обработке заказов и массовых рассылках.
___
## Роли пользователей

- **Администратор**: полный доступ к системе, включая управление пользователями, товарами и заказами.
- **Пользователь**: возможность просматривать товары, добавлять их в корзину, оформлять заказы и оставлять отзывы.
___
## Технологический стек

- **Frontend**: HTML + CSS (без фреймворков).
- **Backend**: Golang.
- **База данных**: PostgreSQL.
- **Брокер сообщений**: RabbitMQ (для обработки асинхронных задач).
- **Контейнеризация**: Docker, Docker Compose.
